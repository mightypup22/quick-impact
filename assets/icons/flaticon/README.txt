Drop-in für Flaticon-Icons
--------------------------
1) Lade passende SVGs von flaticon.com herunter (einheitlicher Stil, z. B. "Lineal", "Bold", "Duotone").
2) Lege sie hier ab und benenne sie exakt wie folgt, damit das HTML nichts ändern muss:
   - trust1.svg, trust2.svg, trust3.svg
   - icon-excel.svg, icon-dashboard.svg, icon-app.svg, icon-qm.svg, icon-controlling.svg
3) Kopiere die Dateien zusätzlich ins Verzeichnis /assets (überschreiben), damit sie live verwendet werden.
4) Falls Attribution erforderlich ist (Flaticon Free), füge im Impressum/Footernachweis hinzu.
   Beispiel (HTML-Kommentar ist bereits im Impressum vorbereitet):
     <!-- Icons von Flaticon: https://www.flaticon.com -->
